# CHANGELOG

## Додано
- Новий файл `AGENTS.md` з архітектурними та технічними правилами для ШІ-агентів.
- Статичні сторінки для Політики конфіденційності (`privacy/index.html`) та Умов використання (`terms/index.html`).
- Банер згоди на використання cookies у `static/js/main.js` та стилі для нього в `static/css/main.css`.
- Глобальний лоадер (`#loader-wrapper`) для всіх сторінок для плавного завантаження.

## Змінено
- Оновлено `static/js/main.js`: додано логіку закриття лоадера та керування банером cookies.
- Оновлено `static/css/main.css`: додано стилі для лоадера, банера cookies та покращено адаптивність.
- Оновлено сторінку акаунта (`account/index.html`) та карту сайту (`sitemap.txt`) для відповідності новій структурі.

## Безпека
- Запроваджено обов'язкове використання банера згоди на cookies.
- Дотримання правил безпеки щодо cookies (HttpOnly, Secure, SameSite), описаних в `AGENTS.md`.
