CraftAntivirus — Антивірус, який сканує за допомогою VirusTotal та ШІ

Про антивірус

CraftAntivirus - це інноваційний антивірусний інструмент, розроблений для забезпечення високого рівня безпеки вашої системи. Він поєднує в собі потужність двох основних технологій: сканування через VirusTotal та аналіз за допомогою штучного інтелекту (ШІ) Gemini.

Завдяки інтеграції з VirusTotal, антивірус може використовувати дані про загрози від багатьох відомих антивірусних двигунів, надаючи точну і всебічну оцінку файлів. Аналіз за допомогою ШІ дозволяє виявляти нові, раніше невідомі загрози, аналізуючи код і поведінку файлів на рівні, недоступному для традиційних методів.

Наш продукт особливо корисний для аналізу файлів з невідомих джерел, потенційно шкідливих скриптів та інших загроз, які можуть бути пропущені звичайними антивірусами.

Встановлення

  • Створіть папку в будь-якому зручному місці.
  • Розпакуйте архів CraftAntivirus.zip в цю папку.
  • Відкрийте файл main.exe, створиться файл config.py.
  • Налаштуйте файл config.py, особливо API-ключі.

    Примітка: API ключ Gemini абсолютно безкоштовний на AI Studio, але не в Vertex AI, там платно.
    Після налаштування config.py можна знову запускати main.exe та перевіряти файли.
